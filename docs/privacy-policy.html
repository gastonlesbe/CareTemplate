Política de Privacidad - FamilyPal

Última actualización: 17 de diciembre de 2024

1. Introducción

FamilyPal ("nosotros", "nuestro" o "la aplicación") se compromete a proteger tu privacidad. Esta Política de Privacidad explica cómo recopilamos, usamos y protegemos tu información cuando usas nuestra aplicación móvil FamilyPal.

2. Información que Recopilamos

2.1 Información que Tú Proporcionas

  • Datos de tus sujetos/miembros de familia: Nombres, fechas de nacimiento, notas y otra información que ingresas sobre tus sujetos
  • Eventos y recordatorios: Información sobre eventos, citas y recordatorios que creas
  • Gastos: Información sobre gastos relacionados con tus sujetos

2.2 Información Recopilada Automáticamente

  • Datos de uso: Información sobre cómo usas la aplicación
  • Identificadores de dispositivo: Para sincronizar tus datos entre dispositivos
  • Datos de Firebase Analytics: Para mejorar la aplicación (anónimos y agregados)

3. Cómo Usamos tu Información

Utilizamos la información recopilada para:

  • Proporcionar y mejorar nuestros servicios
  • Sincronizar tus datos entre dispositivos usando Firebase
  • Analizar el uso de la aplicación para mejoras (datos anónimos)
  • Mostrar anuncios relevantes a través de Appodeal

4. Servicios de Terceros

4.1 Google Firebase

  • Propósito: Almacenamiento y sincronización de datos
  • Datos: Tus eventos, sujetos y gastos se almacenan en Firebase Firestore
  • Autenticación: Usamos autenticación anónima de Firebase
  • Política: Política de Privacidad de Firebase

4.2 Appodeal

  • Propósito: Mostrar anuncios en la aplicación
  • Datos: Pueden recopilar datos de publicidad (ID de publicidad, ubicación aproximada)
  • Política: Política de Privacidad de Appodeal

4.3 Firebase Analytics

  • Propósito: Analizar el uso de la aplicación
  • Datos: Datos anónimos y agregados sobre el uso
  • Política: Política de Privacidad de Google Analytics

5. Almacenamiento de Datos

Tus datos se almacenan de forma segura en:

  • Dispositivo local: Usando Room Database (base de datos local)
  • Nube: Firebase Firestore (servidores de Google)

6. Seguridad

Implementamos medidas de seguridad para proteger tu información:

  • Encriptación de datos en tránsito (HTTPS)
  • Autenticación segura a través de Firebase
  • Acceso restringido a tus datos (solo tú puedes acceder)

7. Tus Derechos

Tienes derecho a:

  • Acceder a tus datos personales
  • Solicitar la eliminación de tus datos
  • Exportar tus datos
  • Retirar tu consentimiento

Para ejercer estos derechos, contáctanos a través de la información de contacto proporcionada.

8. Retención de Datos

Conservamos tus datos mientras uses la aplicación. Puedes eliminar tus datos en cualquier momento desde la aplicación o contactándonos.

9. Cambios a esta Política

Podemos actualizar esta Política de Privacidad ocasionalmente. Te notificaremos de cualquier cambio publicando la nueva política en esta página y actualizando la fecha de "Última actualización".
